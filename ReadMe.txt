DESCRIPTION
------------
The files provided in this project contain the python source code
referent to the algorithms described in the paper “DACCER: Distributed Assessment of the Closeness CEntrality Ranking in complex networks” available at http://www.sciencedirect.com/science/article/pii/S1389128613001412.


REQUIREMENTS
------------
To run the algorithms presented, the following modules are required:
 * NetworkX (https://networkx.github.io)
 * NumPy (http://www.numpy.org)
 * SciPy (http://www.scipy.org)
